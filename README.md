# ComfyUI-Tracking

## Introduction
**ComfyUI-Tracking** This node pack helps to conveniently collect invocation data from workflows for further study.

## How To Install
This node is under development, so it is not yet available on the official node pack site. 

You can download the node pack from the [release page](https://github.com/ComfyUI/ComfyUI-Tracking/releases) and install it manually.
* Navigate to `ComfyUI/custom_nodes` in your terminal (cmd).
* Clone the repository under the `custom_nodes` directory using the following command:
  ```
  git clone https://github.com/WaiyanLing/ComfyUI-Tracking.git comfyui-tracking
  cd comfyui-tracking
  ```
* Install dependencies in your Python environment.
    * For Windows Portable, run the following command inside `ComfyUI\custom_nodes\comfyui-tracking`:
        ```
        ..\..\..\python_embeded\python.exe -m pip install -r requirements.txt
        ```
    * If using venv or conda, activate your Python environment first, then run:
        ```
        pip install -r requirements.txt
        ```
